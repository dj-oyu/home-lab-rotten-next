## ドキュメントの管理方針

1. **更新の徹底**: コードの変更に伴い、関連するドキュメントも必ず更新してください
2. **日本語優先**: 基本的に日本語で記述しますが、技術用語は英語のままでも構いません
3. **明確性**: 誰が読んでも理解できるよう、明確で簡潔な記述を心がけてください
4. **バージョン管理**: 重要な変更は更新履歴として記録してください

## 貢献方法

ドキュメントの改善提案や誤字脱字の修正など、どのような貢献も歓迎します。
